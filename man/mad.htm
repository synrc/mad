MAD

INTRO

MAD module provides user level interface for console commands.

  • main/1 — IO parameters.
  • help/0 — Display help.
  • profile/0 — Current MAD profile.
  • atomize/1 — Commands mappings.

LOOP

main(list(string())) -> integer().

Simple yet understandable command processing and error loop. Function atomize/1 helps to accumulate known agruments as atoms. Function profile/0 retrieves module with MAD API implementation. main/1 function returns error status to OS with halt.
